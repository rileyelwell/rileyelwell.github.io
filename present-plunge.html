Present Plunge

December 2023
Game Jam
Game Design, Pixel-Art, UI/UX
Unity, C#, Aseprite

Description

This project started as a one week game jam during the Christmas season. It was my first jam, and I decided to create everything myself (except the audio files). I then made some slight tweaks and documented my process extensively for my work sample that I used to apply to Futuregames. This was a great learning experience and holds a special place in my heart because it was one of my first projects that really got me started with game design and development.
