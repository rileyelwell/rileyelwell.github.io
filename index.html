Truck Stop

Genre: 2D Infinite Side-Scroller Software: Unity, Aesprite Languages: C# Project Role: Solo Developer

Description: This is a simple 2D infinite side-scroller game where you control a truck that must stop at infinitely-appearing stop signs. There are gameplay elements that include varying truck speeds and road friction to increase difficulty. I developed a pooling system for resetting GameObjects (signs, truck, background, etc.) to optimize the performance. Inspiration: During Summer 2023 I interned at Daimler Truck North America and was inspired by their trucks (and my time there). I wanted to show an original creative product to my co-workers before leaving, so I constructed this project in 4-5 days. I created all assets, except a free truck horn audio clip. The game can be played here: https://rileyelwell.itch.io/truck-stop

Galaga ReInspired

Genre: 2D Space Shooter Software: Unity, Aesprite Languages: C# Project Role: Solo Developer

Description: Inspired by the original Galaga, this is a simple 2D space shooter game. You control a small ship, and your goal is to shoot bullets to hit the ships that infinitely appear. There are three types of ships that all have slightly different movement mechanics. The higher the score the better! The Inspiration: This is my first game I developed and I wanted to start with something simple. I remembered how much I enjoyed playing Galaga as a kid, so I decided recreate elements from that game while using my own ideas too. I used Unity and created the assets myself (except the free space background image), taking about a month to flesh out details. I experimented with bullet pools and spawning ships with different methods to improve performance. The game can be played at: https://rileyelwell.itch.io/galaga-reinspired

GLSL Grass Shader

Genre: Grass Shader, Experimenting Software: Visual Studio Languages: OpenGL/GLSL, C++ Project Role: Solo Developer

Description: During my Computer Graphics Shaders course at Oregon State University, I was tasked with creating a final project experimenting with a shader not explored in the curriculum. I decided on researching and developing a realistic grass shader. The video goes more in depth, but my process included applying methods such as billboarding, adding rotated quads for fuller grass, and Fractal Brownian Motion for a somewhat random color of grass across a period of space. For the wind simulation, I used a noise texture and applied it to the top of the grass for "random" movement.

Object Hunt

Genre: Multiplayer Mobile Game (Android) Software: Android Studio Languages: Kotlin, JS Project Role: Google Maps API Dev / Backend Assister

ObjectHunt is a fun and interactive two-player game that challenges players to find objects based on their name and location. The objective of the game is to be the first player to locate the object and take a picture of it while being in the right location. Each successful capture earns a point, and the game continues until all objects in the submitted queue have been found. To detect objects, ObjectHunt uses Android ML Kit on the frontend and Google's Vision API on the backend. Additionally, to support the app's multiplayer capability, our team wrote a custom backend using Node.js as the framework and Socket.IO as the communication library. Team Members: Adilbek Bazarkulov, Anh Nguyen, Riley Elwell, Imgyeong Lee
